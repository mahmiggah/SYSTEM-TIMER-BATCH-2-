📘 Timer Guide

⏱️ Set Time – hours, minutes, seconds.

🔄 Count Down / Up – toggle direction; the circular ring animates accordingly.

▶ Start / ⏸ Pause – single button toggles between start and pause.

🔄 Reset – resets to the last set duration.

🚩 Markers – add coloured flags (Red, Green, Yellow) on the timeline. When a marker is reached, a toast appears and the flag disappears.

📱 Mobile & Offline – fully responsive; install as PWA for offline use.
