📖 Timer User Guide

  • ⏱️ Set Time – minutes & seconds for countdown or count‑up.
  • ⬇️/⬆️ Count Down/Up – toggle between counting down to zero or up to a target.
  • ▶ Start / ⏸ Pause – starts or pauses the timer.
  • 🔄 Reset – restarts the timer to the last set duration.
  • 🚩 Set Marker – add a flag at a specific time. When reached, it beeps and moves to the "✅ Already reached" list.
  • 🗑️ Clear All Markers – removes both pending and reached markers.
  • 🎨 Color flashes – green at start, yellow at halfway, red at finish.
