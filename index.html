📘 Timer Guide

⏱️ Set Time
Enter hours, minutes, and seconds. Click "Set Timer".

🔄 Count Down / Up
Toggle between counting down to zero or counting up to the target.

▶ Start / ⏸ Pause
Start or pause the timer. Pausing does not save data.

🔄 Reset
Reset the timer to the last set duration (does not affect markers).

🚩 Markers
Add flags at specific times. Choose Red, Green, or Yellow.
• Flags appear on the timeline (colored triangles).
• When a marker is reached, a popup message appears, and the flag disappears.
• Use "Clear All Markers" to remove all pending flags.

📱 Offline & Install
Install this app to your device for offline use and a native-like experience.
