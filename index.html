📖 Timer User Guide

  • ⏱️ Set Time – hours, minutes & seconds for countdown or count‑up.
  • ⬇️/⬆️ Count Down/Up – toggle between counting modes.
  • ▶ Start / ⏸ Pause – controls the timer.
  • 🔄 Reset – restarts timer to last set duration.
  • 🚩 Set Marker – add a flag; when reached, it beeps and moves to "Already reached".
  • 🗑️ Clear All Markers – removes all markers.
  • 🎨 Color flashes – green at start, yellow at halfway, red at finish.
