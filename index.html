📘 Timer Guide

▶ Start / ⏸ – starts/pauses the timer. If preparation time is set, it runs first.

🔄 Reset – resets to last set duration (cancels preparation).

☰ Settings – set time, toggle mode, enable continue past zero, set preparation time, manage traffic light events, and rename timer.

🚦 Events – add times (in seconds) when the traffic light changes colour (e.g., 15s green, 10s yellow, 5s red).

✏️ Timer name – click the name on the main screen or edit in Settings.

⌨️ Hotkeys – Space Start/Pause, R Reset, S Settings, E Add Event.
